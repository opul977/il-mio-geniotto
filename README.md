# Il Mio Geniotto 🤖✨

**Il Mio Geniotto** è un'applicazione web moderna e coinvolgente progettata per aiutare studenti (bambini e ragazzi) a svolgere i compiti in modo interattivo e divertente utilizzando l'intelligenza artificiale di Google Gemini.

## Caratteristiche 🚀

- **Chat con Geniotto**: Un assistente IA che adatta il tono in base al livello scolastico.
- **Supporto Foto**: Carica la foto di un compito e ricevi la spiegazione passo-passo.
- **Design Ultra-Playful**: Interfaccia colorata, animata e facile da usare.
- **Multilivello**: Supporto specifico per Scuola Primaria, Media e Superiore.

## Configurazione Locale 🛠️

1. Clona il repository.
2. Installa le dipendenze: `npm install`.
3. Crea un file `.env` e aggiungi la tua chiave API:
   ```env
   GEMINI_API_KEY=tua_chiave_qui
   ```
4. Avvia il server: `npm run dev`.

## Deploy su Vercel 🌐

Il modo più semplice per migrare il sito è usare Vercel:

1. Carica il codice su un repository **GitHub**.
2. Connetti GitHub a **Vercel**.
3. Nelle impostazioni del progetto su Vercel, aggiungi la variabile d'ambiente `GEMINI_API_KEY`.
4. Vercel effettuerà il deploy automatico a ogni aggiornamento.
